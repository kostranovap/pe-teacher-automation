Импорт студентов из Excel

Инструкция по импорту

Подготовьте Excel файл (.xlsx или .xls) со следующими столбцами:

  1. ФИО - полное имя студента
  2. Номер студ. билета - уникальный номер
  3. Группа - название существующей группы
  4. Пол - М/Ж или male/female
  5. Дата рождения - в формате даты (необязательно)
  6. Мед. группа - основная/подготовительная/СМГ/освобождение (необязательно)
